# 診断ロジック確認メモ

この版では「OS希望」を強く尊重するように修正しました。

## 確認済みパターン

| 入力パターン | 期待される着地 |
|---|---|
| ノートPC / Mac / ブログ | MacBook Air バランス型 |
| ノートPC / Mac / ChatGPT・SNS | MacBook Air バランス型 |
| ノートPC / Mac / 動画編集 | Mac クリエイター型 |
| ノートPC / Mac / プログラミング | MacBook Air バランス型 |
| デスクトップPC / Mac / 家用 | Mac mini / 据え置きMac型 |
| Mac / FX・MT4 | Mac希望・用途注意型 |
| Mac / ゲーム | Mac希望・用途注意型 |
| デスクトップPC / Windows / 家用ブログ | 家用デスクトップPC コスパ型 |
| デスクトップPC / Windows / 動画編集 / 高予算 | デスクトップPC 高性能・拡張型 |
| ノートPC / Windows / ブログ | 副業・ブログ用 地雷回避Windows型 |

## 方針

- Macを選んだら、基本的にWindows診断結果へ落とさない。
- Mac + FX/MT4 または Mac + ゲームは、Windowsを強制提案せず「Mac希望・用途注意型」にする。
- デスクトップ + Mac は、Windowsデスクトップではなく「Mac mini / 据え置きMac型」にする。
- デスクトップ + Windows は、前回どおりデスクトップ型にする。
