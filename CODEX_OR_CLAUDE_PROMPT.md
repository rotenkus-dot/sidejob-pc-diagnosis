あなたはNext.js/TypeScriptの実装担当です。
このリポジトリは「副業・動画編集・AI初心者向け｜地雷PC回避診断」のMVPです。

## 現在の状態

- Next.js 15 / React 19 / TypeScript
- `npm run build` は成功済み
- 診断フォーム、結果ページ、診断タイプDB、救済ルール、地雷回避リスト、アフィ検索リンク生成を実装済み

## 目的

このサイトをVercelにそのまま公開できる品質に仕上げてください。

## 絶対に守るコンセプト

これは「おすすめPCランキング」ではありません。
初心者が地雷PCを買わないための、予算別PC診断です。

結果ページでは、特定商品ではなく以下を出します。

1. 買っていい条件
2. 避けたい条件
3. 最大2つの現実的な警告/逃げ道
4. 地雷PCチェックリスト
5. ECサイトで探す検索条件
6. 周辺機器・副業ツールのクロスセル

## 注意

- 警告は最大2つまでというUXルールを崩さない
- 特定メーカーを根拠なく批判しない
- 「絶対に壊れる」「必ず稼げる」など断定表現を避ける
- 広告・アフィリエイト表記は消さない
- Amazon/楽天/Yahooの正式なアフィリエイトURL形式は、運営者がASP画面で最終確認する前提
